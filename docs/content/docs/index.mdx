To get started with Toolsync, you can either use the [create command](/docs/create) (described below) or install it manually. The create command is the recommended way to get started, as it will set up a new project with all the necessary configuration in no time. It works for both **new and existing projects**.

## Setup toolsync

<Callout>We recommend using [pnpm](https://pnpm.io) in the examples below.</Callout>

<PackageManagerTabs>

<Tab value='pnpm'>
  
```shell
pnpm create @toolsync
```

</Tab>
<Tab value='npm'>

```shell
npm init @toolsync
```

</Tab>
<Tab value='yarn'>

```shell
yarn create @toolsync
```

</Tab>

</PackageManagerTabs>

## Manual Installation

First of all, you need to install the [Toolsync CLI](/docs/cli). It's also recommended you install the [@toolsync/builtin](/docs/builtin) package. You can do this with your package manager of choice:

<PackageManagerTabs>

<Tab value='pnpm'>
  
```shell
pnpm add -Dw @toolsync/cli @toolsync/builtin
```

</Tab>
<Tab value='npm'>

```shell
npm install --save-dev @toolsync/cli @toolsync/builtin
```

</Tab>
<Tab value='yarn'>

```shell
yarn add --save-dev @toolsync/cli @toolsync/builtin
```

</Tab>

</PackageManagerTabs>

## Create config files

Now that Toolsync is installed and configured, you can run `toolsync prepare`.

<PackageManagerTabs>
<Tab value='pnpm'>

```shell
pnpm toolsync prepare --config toolsync.json
```

</Tab>
<Tab value='npm'>

```shell
npx toolsync prepare --config toolsync.json
```

</Tab>
<Tab value='yarn'>

```shell
yarn toolsync prepare --config toolsync.json
```

</Tab>
