The @toolsync/builtin package provides a set of built-in tools that are available for use in your Toolsync project.

## Installation

<PackageManagerTabs>

<Tab value='pnpm'>
  
```bash
pnpm add -Dw @toolsync/builtin
```

</Tab>
<Tab value='npm'>

```bash
npm install --save-dev @toolsync/builtin
```

</Tab>
<Tab value='yarn'>

```shell
yarn add --save-dev @toolsync/builtin
```

</Tab>

</PackageManagerTabs>
